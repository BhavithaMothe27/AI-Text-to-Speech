🔊 AI Text-to-Speech

An AI-powered Text-to-Speech (TTS) application that converts written text into natural-sounding speech using Google Text-to-Speech (gTTS). The project provides a simple and interactive Gradio web interface for generating audio from text.

🚀 Features

- 📝 Convert text into speech
- 🔊 Generate MP3 audio output
- 🌍 Supports English language
- ⚡ Fast speech generation
- 🎨 Interactive Gradio interface

🛠️ Technologies Used

- Python
- gTTS (Google Text-to-Speech)
- Gradio

📦 Installation

```bash
pip install -r requirements.txt
```

▶️ Run the Application

```bash
python app.py
```

🔄 Workflow

1. User enters text.
2. The application sends the text to gTTS.
3. gTTS converts the text into speech.
4. The generated speech is saved as an MP3 file.
5. The audio is played through the Gradio interface.
